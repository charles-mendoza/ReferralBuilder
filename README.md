# Referral Builder

A Referral Builder app made with:
- React
- TypeScript
- Vite
- SASS
- Storybook
- Supabase

## Deployment

To run app in local:
```
npm install
npm run dev
```

To run Storybook in local:
```
npm run storybook
```
